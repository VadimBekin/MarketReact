# MarketReact - Интернет-магазин на React
Функциональное приложение интернет-магазина с каталогом товаров и корзиной покупок.

# deploy:
https://vadimbekin.github.io/MarketReact/


# Запуск приложения:
# 1. Клонировать репозиторий
git clone https://github.com/VadimBekin/MarketReact.git
cd MarketReact/my-app
# 2. Установить зависимости
npm install
# 3. Запустить проект в режиме разработки
npm run dev


# Основные функции:
*Каталог товаров:
    "Пагинация (12 товаров на странице)"
    "Сортировка по цене, названию, рейтингу"
    "Фильтрация по категориям"
    "Адаптивная сетка товаров"
*Корзина покупок:
    "Добавление и удаление товаров"
    "Изменение количества"
    "Расчет общей суммы"
    "Сохранение в LocalStorage"
*Умный поиск:
    "Поиск по названию, описанию, категории"
    "Debounce 500ms для оптимизации"
    "Поиск по Enter или кнопке"
*Адаптивный дизайн.
*Toast-уведомления.


# Основные технологии:
React + TypeScript
Vite - сборщик проекта
React Router - маршрутизация
CSS Modules - стилизация
LocalStorage - хранение данных корзины
Fake Store API - тестовые данные товаров
